# SauceDemo – Dokumentacja Testowa

## Opis projektu

Projekt został wykonany samodzielnie w celu rozwijania umiejętności z zakresu testowania manualnego aplikacji webowych.

Celem projektu było przygotowanie kompletnej dokumentacji testowej oraz przeprowadzenie testów funkcjonalnych aplikacji *SauceDemo*.

---

# Zakres projektu

## Testowany obszar

Przeprowadzone testy obejmowały:

- logowanie użytkownika,
- listę produktów,
- sortowanie produktów,
- dodawanie produktów do koszyka,
- usuwanie produktów z koszyka,
- podgląd koszyka,
- proces składania zamówienia,
- walidację formularzy,
- komunikaty o błędach.
